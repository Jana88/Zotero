Zotero & Reference Manager

Weet-me-nietjes, 12 maart 2014

Op het vlak van werken met programma's voor het beheer van referenties, heb ik al eerder verworven kennis kunnen toepassen. Tijdens mijn vorige studies, namelijk in de verpleegkunde, heb ik reeds gewerkt met Reference Manager. Nu, in de huidige taal- en regiostudie: Japanologie, wordt het gebruik van Zotero, eerder dan Reference Manager, aangeraden. Het gebruiken van dergelijke tools is handig en bespaart je tijd. Tijd die je anders aan de lay-out zou besteden van je bibliografie. Vooral als je met verwijzingen in de tekst werkt, bijvoorbeeld onder de vorm van nummers, moet je niet elke keer de nummering, verwijzend naar je bronnen, aanpassen als je halverwege de tekst plots een bron zou bijvoegen. Reference Manager paste deze nummers immers automatisch aan zodat de correct volgorde behouden bleef.

Met Zotero heb ik echter nog niet veel ervaring. Het gebruik van zowel Zotero als Reference Manager ligt vrij dicht bij elkaar, wat het voor mij makkelijker maakt om met Zotero te leren werken. Wat me wel is opgevallen is dat je bij Zotero een functie hebt die je toelaat om bij elke toegevoegde bron ook aantekeningen toe te voegen. Dit is vooral handig als ik beter wil weten wat er precies van inhoud in de bron te vinden is, of als ik bijvoorbeeld een inhoudstafel wil toevoegen. Ook is me opgevallen dat je bij Zotero 'tags' kunt toevoegen, een zeer handig hulpmiddel indien je snel je bronnen wilt terugvinden.

Een nadeel echter is dat je met Zotero geen citaties kunt exporteren van google books, een website die ik regelmatig gebruik. Vanuit google books kun je dit enkel doen voor programma's zoals RefMan (Reference Manager), EndNote, en BibTeX.
